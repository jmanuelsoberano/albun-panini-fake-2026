# UX audit vigente

Producto: Álbum Mundial 2026.

## Hallazgos principales

1. La app debe entrar como producto final, no como demo técnica.
2. Sin sesión, el usuario debe poder explorar en solo lectura y recibir un CTA claro de login.
3. La colección necesita convivir con una capa fuerte de torneo: grupos, partidos, marcadores,
   goleadores, sedes y llaves.
4. Todo dato deportivo visible debe ser real o quedar marcado como pendiente.
5. El diseño debe sentirse más moderno, deportivo e interactivo.

## Requisitos de contenido

- 48 selecciones reales.
- Plantillas reales de 26 jugadores.
- 16 sedes reales.
- Grupos A-L reales.
- 72 partidos de fase de grupos.
- Ronda de 32, octavos, cuartos, semifinales, tercer lugar y final.
- Marcadores y goleadores solo con verificación.

## Gate de calidad

- Sin textos técnicos visibles sobre Firebase/modo local.
- Sin datos deportivos sin verificar.
- Sin logos, escudos, fotos ni arte protegido.
- UI responsive en móvil y escritorio.
- Navegación clara entre inicio, colección, torneo, equipos, sedes, sala y retos.
- Smoke `real-worldcup-content` debe pasar.
